# Sandboxing & Isolation

<TagList tags={frontmatter.tags} />
<AttributionList contributors={frontmatter.contributors} />

This section is the **canonical home** for guidance on sandboxing, isolation, and containment as part of DevSecOps.

Sandboxing and isolation are *implementation controls* used to reduce blast radius when running untrusted or semi-trusted code and workflows (CI steps, build tooling, plugins, scripts, AI tool calls, etc.).
