# AI Security

<TagList tags={frontmatter.tags} />
<AttributionList contributors={frontmatter.contributors} />

AI systems create a new security surface: they process untrusted input, make autonomous decisions, and can trigger
high-impact actions through tools and integrations. This framework covers practical controls for running AI systems
safely in production.
